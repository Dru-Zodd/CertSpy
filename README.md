# CertSpy
---
## Description
CertSpy is a powerful OSINT and legal pentesting tool that leverages the crt.sh API to uncover hidden subdomains and enhance your security assessments.

## Installation & Usage
1. Clone the repo.
2. cd /CertSpy
3. python3 Certspy.py Example.com
